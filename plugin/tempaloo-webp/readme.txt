Convert your WordPress images to WebP and AVIF automatically. Lighter pages, faster loads, no setup.

== Description ==

Tempaloo WebP converts your images to modern formats (WebP, and AVIF on paid plans) via our API, and serves the optimized version to browsers that support it. No local tooling, no server-side libraries to install.

* Automatic conversion on upload
* Serve WebP/AVIF to supported browsers (falls back gracefully)
* Per-image quality control
* Free plan: 150 images / month
* Paid plans starting at €5/month — see https://tempaloo.com/webp

== Installation ==

1. Install and activate the plugin.
2. Go to **Settings → Tempaloo WebP**.
3. Click **Generate an API key** to create a free account, or paste an existing key.
4. Click **Activate**. That's it.

== Frequently Asked Questions ==

= Do I need an account? =
Yes. Conversion happens on our API, so you need a free or paid license key.

= What happens if I hit my quota? =
New uploads stop being converted until the next billing cycle or an upgrade. Existing images stay served as WebP.
